# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - Initial Archival Release

### Added
- FastAPI-based CV screening application
- Three string-matching algorithms: Brute Force, Rabin-Karp, KMP
- Web dashboard with TailwindCSS and Chart.js
- Batch CV analysis with performance metrics
- PDF and DOCX document parsing
- Three predefined job descriptions (AI/ML Engineer, Data Scientist, Full-Stack Developer)
- Algorithm performance comparison and visualization
- Candidate ranking by match percentage

### Technical Details
- Python 3.8+ support
- FastAPI web framework with Uvicorn server
- PyPDF2 for PDF text extraction
- python-docx for Word document parsing
- Interactive web UI with real-time analysis
